# Features of Redmine Pivot Plugin

Here are the key features of this plugin:

1.  **Interactive Pivot Table & Charts**
    *   Analyze Redmine issues dynamically using drag-and-drop pivot tables.
    *   Supports various visualizations including Tables, Heatmaps, Bar charts, Line charts.

2.  **Persistent Configuration**
    *   Save your favorite pivot table layouts (Rows, Columns, Aggregators) and filter settings to the server.
    *   Restored settings are shared via unique URLs, making it easy to collaborate with team members.

3.  **Customizable Field Selection**
    *   Easily toggle the visibility of specific fields using checkboxes to focus on the data that matters.
    *   Unused fields are neatly organized to keep the interface clean.

4.  **Multi-Language Support (English/Japanese)**
    *   Fully localized for both English and Japanese environments, including aggregator names and UI elements.

5.  **Automatic Date Grouping** 
    Date fields are automatically broken down for detailed time-series analysis:
        *   **Year**: Analyze broad trends.
        *   **Month**: Track seasonal or monthly progress.
        *   **Day**: Monitor daily activity.
        *   **Year-Month**: View monthly data across years.

6.  **Regex Text Grouping**
    *   Create new attributes by extracting patterns from text fields (e.g., extracting tags from subject lines) using regular expressions.

# 機能一覧

本プラグインの主な機能は以下の通りです：

1.  **インタラクティブなピボットテーブルとチャート**
    *   ドラッグ＆ドロップ操作でRedmineのチケットを動的に分析できます。
    *   表形式、ヒートマップ、棒グラフ、折れ線グラフなど、多彩な可視化に対応しています。

2.  **設定のサーバー保存機能**
    *   作成したピボットテーブルのレイアウト（行、列、集計方法）やフィルタ設定をサーバーに保存できます。
    *   保存した設定はURLで共有できるため、チームメンバーと同じ分析結果を簡単に共有可能です。

3.  **表示項目の選択カスタマイズ**
    *   チェックボックスを使って、分析に必要なフィールドだけを簡単に表示・非表示に切り替えられます。
    *   使用しないフィールドは整理され、画面がすっきりと保たれます。

4.  **多言語対応（英語・日本語）**
    *   英語と日本語の完全なローカライズに対応しており、集計方法の名称なども日本語で利用可能です。

5.  **日付の自動グループ化**
    * 日付項目は時系列分析のために自動的に以下の単位に分解されます：
        *   **年**: 長期的な傾向を分析できます。
        *   **月**: 月ごとの進捗や傾向を追跡できます。
        *   **日**: 日次の活動状況をモニタリングできます。
        *   **年月**: 年を跨いだ月ごとのデータを一目で確認できます。

6.  **正規表現によるテキストグループ化**
    *   テキストフィールド（件名など）から正規表現パターンに一致する部分を抽出し、新しい属性として集計に使用できます。
